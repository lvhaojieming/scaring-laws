# Scaring Laws

一个面向旅行者、内容创作者和研究者的“全球高风险法律速查”项目种子版本。

当前仓库已经从空仓库搭成了一版可直接打开的静态首页，目标不是一次把功能做满，而是先把这个项目最关键的三件事落地：

1. 先定义信息结构
2. 先确定视觉方向
3. 先让仓库可以直接展示和继续扩展

## 当前包含

- 一个零依赖静态首页
- 可筛选的法律卡片面板
- 聚焦详情区，便于后续演化为数据库或国家页
- 项目研究方法区块
- 适合直接部署到 GitHub Pages / Netlify / Vercel 的目录结构

## 目录结构

```text
.
├── data/
│   └── laws.js
├── index.html
├── script.js
├── styles.css
└── README.md
```

## 本地预览

这版不依赖 Node，可以直接用任意静态服务器预览。

如果本机有 Python 3：

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://localhost:4173
```

## 当前假设

因为仓库里还没有既定技术栈、页面结构或接口设计，这一版先采用了最轻量的静态站方案。这样做有几个好处：

- 不会被本地缺失的 Node 环境卡住
- 仓库可以马上有一个可展示的项目首页
- 后续无论你想继续做内容站，还是迁移到 React / Next.js，都不会浪费这轮结构设计

## 下一步建议

- 把 `data/laws.js` 继续补成更完整的种子数据
- 给每条卡片补上官方来源链接字段
- 增加搜索、标签和国家页
- 数据量上来后，再迁移到 JSON、CMS 或数据库

## 提醒

当前卡片内容主要用于展示项目结构与交互方向，不构成法律意见。正式发布前，建议对每个条目补充官方法条、政府说明或律师核验来源。
